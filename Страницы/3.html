Теги, которых не было в видео

Дополнительная информация для тех, кто хочет знать больше.
  • <br>
  • <i>
  • <em>
  • <strong>
  • <b>
  • <small>

Тег <br>

Элемент <br> (от англ. line break — перенос строки) устанавливает перевод строки в том месте, где он находится.
В отличие от элемента <p>, использование <br> не добавляет пустой отступ перед строкой.
<p> используется для обозначения параграфа (абзаца) текста,
в то время как <br> - для создания отступа между строками/переноса текста на другую строку.

Тег <i>

Устанавливает курсивное начертание шрифта. Допустимо использовать этот тег совместно с другими тегами, которые определяют начертание текста.

Тег <em>

Элемент <em> предназначен для слов, которые имеют подчёркнутый акцент по сравнению с окружающим текстом, который часто ограничивается словом или словами предложения и влияет на смысл самого предложения. Обычно этот элемент отображается курсивом.
Тег <i> и тег <em> как раз такие теги. Внешне ничем не различаются, но семантический смысл разный. Первый тег <i> просто выделяет текст курсивным начертанием, второй же тег <em> придает тексту, помещенному в него, особый смысл.
Текст становится более важным на странице.

Тег <strong>

<strong> - элемент выделения важного текста.
Элемент сильной значимости (<strong>) указывает на то, что его содержание имеет большое значение, серьёзность или срочность.
Браузеры обычно отображают содержимое жирным шрифтом.
Потоковой контент, Фразовый контент, явный контент.

Тег <b>

Устанавливает жирное начертание шрифта. Допустимо использовать этот тег совместно с другими тегами, которые определяют начертание текста.
Тег <strong> определяет важность отмеченного текста. Тег <b> предназначен для выделения текста без придания ему особой важности.
Визуально оба тега одинаковы, они выделяют текст полужирным.

Тег <small>

Тег <small> уменьшает размер шрифта на единицу по сравнению с обычным текстом.
В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3.
Таким образом, добавление тега <small> уменьшает текст на одну условную единицу.
